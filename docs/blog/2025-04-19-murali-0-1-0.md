Murali v0.1.4 is the first public release of a Rust-powered math animation engine built on wgpu.

{/* truncate */}

This release establishes the core foundation: a world-space rendering pipeline, an embedded Typst/LaTeX backend, a time-driven animation system, and a growing library of tattvas (shapes, text, composites).

## What's in v0.1.4

- **wgpu rendering backend** — cross-platform via Vulkan, Metal, and DX12
- **Tattvas** — Circle, Square, Rectangle, Ellipse, Line, Polygon, Axes, Label, Latex, and more
- **Timeline animations** — move, fade, scale, rotate, morph, write/reveal
- **Easing functions** — 8 built-in easing curves
- **LaTeX & Typst** — embedded compilation, SVG → RGBA rasterization
- **Video export** — offscreen rendering to frames/mp4
- **Camera** — orbit and pan/zoom controllers
