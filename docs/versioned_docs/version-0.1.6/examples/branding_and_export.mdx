# Branding And Export

This section focuses on examples that are useful when you want release-quality outputs rather than just API exploration.

## Recommended Examples

- `murali_logo`
  Focus: visual/reference logo study built from authored geometry
- `murali_logo_transparent`
  Focus: transparent PNG export of the M-style logo with simple example-level toggles
- `map_projection_morph`
  Focus: a polished visual demo with a strong presentation surface
- `fourier_formula_trace`
  Focus: advanced-demo pacing and showcase-style composition

## Good First Pick

Start with `murali_logo_transparent` if your immediate goal is export workflow.

## What To Look For

- title and composition balance
- whether the scene looks good without extra explanation
- whether the example feels release-ready, not just technically correct
- whether the export-oriented example is clearly separated from the visual study example

## Run Locally

```bash
cargo run --example murali_logo
cargo run --example murali_logo_transparent
```
