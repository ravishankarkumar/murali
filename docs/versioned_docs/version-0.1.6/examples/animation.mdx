# Animation

This section is for examples that primarily teach motion and timing.

## Recommended Examples

- `motion_basics`
  Focus: `move_to`, `scale_to`, `rotate_to`, `fade_to`, and readable easing
- `text_animation`
  Focus: `draw`, `undraw`, `typewrite_text`, `reveal_text`, and `indicate`
- `equation_and_matrix_animation`
  Focus: semantic math transforms instead of generic geometric motion
- `traced_paths`
  Focus: motion that leaves behind meaningful geometry

## Good First Pick

Start with `motion_basics` if you want the clearest overview of Murali's core animation verbs.

## What To Look For

- whether the scene teaches one motion idea at a time
- whether the timing is calm enough to read
- whether the reveal order helps the animation feel intentional

## Run Locally

```bash
cargo run --example motion_basics
```
